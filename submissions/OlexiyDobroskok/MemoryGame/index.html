Memory Game
La Casa de Papel

Find out which Money Heist Character you would be!

On the field of twelve cards, find pairs of identical cards with minimal number of attempts.The number of times you try will show what character you are:

  • (Attempts ≤ 12)

    Professor

  • (12 < Attempts ≤ 15)

    Alisia

  • (15 < Attempts ≤ 18)

    Berlin

  • (18 < Attempts ≤ 23)

    Tokio

  • (23 < Attempts ≤ 27)

    Denver

  • (Attempts ≥ 28)

    Arturo
